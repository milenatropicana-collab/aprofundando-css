Os principais tipos de visualização no CSS são definidos pela propriedade display, que determina como os elementos se comportam no layout. valores essenciais incluem block (ocupa a largura total), inline (ocupa apenas o conteúdo), inline-block (combina ambos), flex (layout flexível) e grid (layout em grade).
