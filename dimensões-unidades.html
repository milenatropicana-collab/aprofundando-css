Dimensões e Unidades Fundamentais no css

As dimensões no CSS definem o tamanho de elementos (width, height, padding, etc.) usando unidades absolutas (fixas, ex: px, cm) ou relativas (flexíveis, ex: em, rem, %, vw, vh). Unidades como rem e % são preferidas para responsividade, enquanto px é ideal para elementos que não devem mudar de tamanho.
