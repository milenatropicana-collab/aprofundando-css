Pseudo-classe em Html

Uma pseudo-classe CSS é uma palavra-chave adicionada a um seletor para estilizar um estado especial de um elemento, como :hover (mouse sobre), :focus (foco), ou :nth-child() (posição). Ela permite aplicar estilos dinâmicos baseados na interação do usuário ou estrutura do DOM, sem alterar o HTML, sempre precedida por dois.
